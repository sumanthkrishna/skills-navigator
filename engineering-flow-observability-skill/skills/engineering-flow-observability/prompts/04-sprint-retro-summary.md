# Prompt — Sprint Retro Summary

Using all available story flow reports for sprint `{SPRINT_NAME}`, generate a sprint retro summary.

Include:

1. Total stories analyzed
2. Average commit-to-deploy elapsed time
3. Top three delay categories
4. Repeated CI/CD failures
5. Longest PR approval waits
6. Longest deployment waits
7. Recommended system-level improvements

Use positive, improvement-focused language.
