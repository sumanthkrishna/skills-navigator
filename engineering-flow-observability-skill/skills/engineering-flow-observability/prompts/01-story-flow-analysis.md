# Prompt — Story Flow Analysis

Analyze the engineering flow for Jira story `{JIRA_STORY_ID}`.

Use the available timeline data from Jira, GitHub or Bitbucket, Jenkins, SonarQube, and Spinnaker.

Return:

1. One-paragraph executive summary
2. Timeline table
3. Time spent by category
4. Bottleneck classification
5. Recommended actions
6. Any missing data or low-confidence correlations

Use delivery-system language. Do not blame individual engineers.
